# 🛒 ErikStore – Modern E-commerce Platform

## 🧾 Overview

ErikStore is a modern e-commerce application built to simulate a real-world online shopping experience.

The project focuses on building scalable frontend architecture, handling complex UI states, and delivering a seamless user experience.

## ⚙️ Tech Stack

**Frontend**

* Next.js
* React (Hooks)
* JavaScript
* SCSS (SASS)

**Forms & Validation**

* Formik
* Yup

**Integrations**

* Stripe (test mode)
* Headless CMS (Strapi / mock API)

---

## ✨ Features

* 🛍️ Product catalog with dynamic rendering
* 🔎 Search and filtering system
* 📄 Product detail pages
* 🛒 Shopping cart with persistence
* 💳 Checkout flow with Stripe integration
* ✅ Form validation using Formik + Yup
* ⚡ Loading states and error handling
* 📱 Fully responsive design
